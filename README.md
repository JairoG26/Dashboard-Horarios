# Dashboard-Horarios
Este proyecto cuenta con una versión con datos ficticios de un proyecto con datos confidenciales

## Procesamiento de datos
Se hizo a partir de un dataset creado con un script de python que genera datos random.

### Carga de datos en postgresql
Para cargar los datos se utilizó postgresql, por lo que es necesario configurarlo. Así como configurar los datos del URI en el archivo config.py.

## Tablero de control
Se muestran las páginas del tablero con los nuevos datos.
